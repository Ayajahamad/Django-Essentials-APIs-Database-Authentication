Django QuerySet :

A QuerySet is a collection of data from a database. A QuerySet is built up as a list of objects. QuerySets makes it easier to get the data you actually need, by allowing you to filter and order the data at an early stage. In this tutorial we will be querying data from the Member table.

  • use the .all() method to get all the records and fields of the Member model
  • The values() method allows you to return each object as a Python dictionary, with the names and values as key/value pairs
  • The values_list() method allows you to return only the columns that you specify Ex : mydata = Member.objects.values_list('firstname')
  • You can filter the search to only return specific rows/records, by using the filter() method. Return only the records where firstname is 'Emil' : mydata = Member.objects.filter(firstname='Emil').values()

  • filter() Method :

    The filter() method takes the arguments as **kwargs (keyword arguments), so you can filter on more than one field by separating them by a comma : mydata = Member.objects.filter(lastname='Refsnes', id=2).values()
